# 💻 みんなのスクラッチ作品ギャラリー

スクラッチで作ったゲームや作品をみんなで見られるサイトです！

---

## 🚀 作品のアップロード方法

### ステップ1：スクラッチから作品をダウンロードする

1. スクラッチエディタを開く
2. 左上「**ファイル**」→「**コンピューターに保存する**」をクリック
3. `○○.sb3` というファイルがダウンロードされる

### ステップ2：ファイルの名前を変える ⭐ 大事！

ダウンロードした sb3 ファイルの名前を次のルールで変えてください：

```
なまえ_さくひんのタイトル.sb3
```

**例：**
| 変える前 | 変えた後 |
|---------|---------|
| project.sb3 | たろう_ねこのゲーム.sb3 |
| scratch.sb3 | はなこ_うちゅうぼうけん.sb3 |

> アンダースコア `_` で「なまえ」と「タイトル」を区切ってください

###　TurboWarpを使って、HTML化する　###
.sb3ファイルを読み込み、HTMLにパッケージする

ファイルの名前を変える ⭐ 大事
| 変える前 | 変えた後 |
|---------|---------|
| たろう_ねこのゲーム.sb3 | たろう_ねこのゲーム.html |

###　スクリーンショットを撮って、サムネイルを作る　###
ファイルの名前を変える ⭐ 大事
たろう_ねこのゲーム.png

### ステップ3：GitHubにアップロードする

1. このリポジトリを GitHub で開く
2. `uploads` フォルダをクリック
3. 右上の「**Add file → Upload files**」をクリック
4. sb3 ファイルをドラッグ＆ドロップ
5. 下の「**Commit changes**」ボタンを押す

### ステップ4：自動でサイトに反映される 🎉

アップロードが終わると、GitHub Actions が自動で：
- sb3 を `works/` フォルダにコピー
- ギャラリー一覧（works.json）を更新

だいたい **1〜2分** でサイトに反映されます！

## 📁 フォルダ構成

```
scratch-stage/
├── index.html          ← ギャラリーページ（触らなくてOK）
├── player.html         ← 作品プレイヤーページ（触らなくてOK）
├── works.json          ← 作品リスト（自動更新）
├── uploads/            ← ここにsb3をアップロードする
│   └── たろう_ねこのゲーム.sb3
├── works/              ← コピーされた作品（自動生成）
│   └── たろう_ねこのゲーム.sb3
└── .github/workflows/  ← 自動化の設定（触らなくてOK）
```

---

## ⚙️ 初期セットアップ（先生向け）

### GitHub Pages を有効にする

1. リポジトリの「**Settings**」を開く
2. 左メニューの「**Pages**」をクリック
3. Branch を「**main**」、フォルダを「**/ (root)**」に設定
4. 「**Save**」を押す

`https://あなたのID.github.io/scratch-stage/` でアクセスできます！
