# 🎮 みんなのスクラッチ作品ギャラリー

スクラッチで作ったゲームや作品をみんなで見られるサイトです！

---

## 🚀 作品のアップロード方法

### ステップ1：スクラッチから作品をダウンロードする

1. スクラッチエディタを開く
2. メニュー（ファイル）→「**HTMLとして保存**」をクリック
3. `○○.zip` というファイルがダウンロードされる

### ステップ2：ファイルの名前を変える ⭐ 大事！

ダウンロードした zip ファイルの名前を次のルールで変えてください：

```
なまえ_さくひんのタイトル.zip
```

**例：**
| 変える前 | 変えた後 |
|---------|---------|
| project.zip | たろう_ねこのゲーム.zip |
| scratch.zip | はなこ_うちゅうぼうけん.zip |

> アンダースコア `_` で「なまえ」と「タイトル」を区切ってください

### ステップ3：GitHubにアップロードする

1. このリポジトリを GitHub で開く
2. `uploads` フォルダをクリック
3. 右上の「**Add file → Upload files**」をクリック
4. zip ファイルをドラッグ＆ドロップ
5. 下の「**Commit changes**」ボタンを押す

### ステップ4：自動でサイトに反映される 🎉

アップロードが終わると、GitHub Actions が自動で：
- zip を展開して `works/` フォルダに配置
- ギャラリー一覧（works.json）を更新
- サイトに表示

だいたい **1〜2分** でサイトに反映されます！

## 📁 フォルダ構成

```
scratch-gallery/
├── index.html          ← ギャラリーページ（触らなくてOK）
├── works.json          ← 作品リスト（自動更新）
├── uploads/            ← ここにzipをアップロードする
│   └── たろう_ねこのゲーム.zip
├── works/              ← 展開された作品（自動生成）
│   └── たろう_ねこのゲーム/
│       └── index.html
└── .github/workflows/  ← 自動化の設定（触らなくてOK）
```

## ⚙️ 初期セットアップ（先生向け）

### 1. リポジトリを作成

```bash
# GitHubで新しいリポジトリを作成後
git clone https://github.com/あなたのID/scratch-gallery.git
cd scratch-gallery
```

### 2. GitHub Pages を有効にする

1. リポジトリの「**Settings**」を開く
2. 左メニューの「**Pages**」をクリック
3. Source を「**Deploy from a branch**」に設定
4. Branch を「**main**」、フォルダを「**/ (root)**」に設定
5. 「**Save**」を押す

しばらくすると `https://あなたのID.github.io/scratch-gallery/` でアクセスできます！

### 3. （任意）リポジトリをパブリック or プライベートにする

- **パブリック**：誰でも見られる（GitHub Pages 無料）
- **プライベート**：GitHub Pro/Team が必要（有料）

## ❓ よくある質問

**Q: サムネイル（アイキャッチ）画像は使えますか？**
A: zip の中に `thumbnail.png` または `screenshot.png` という画像ファイルを入れておくと自動で使われます。

**Q: 作品を削除したいです**
A: `works/` フォルダから該当フォルダを削除し、`works.json` から該当エントリを削除してください。

**Q: 作品名や作者名を後から変えたいです**
A: `works.json` を直接編集してください。
